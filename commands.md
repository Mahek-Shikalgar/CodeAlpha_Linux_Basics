## üìÇ Commands Executed

### 1. `pwd`
**Description:**  
Displays the present working directory (current location in the file system).

**Purpose:**  
Used to verify the current directory before performing file or folder operations.

---

### 2. `ls`
**Description:**  
Lists all files and directories in the current directory.

**Purpose:**  
Used to check existing files and folders before creating new ones.

---

### 3. `mkdir devops_task`
**Description:**  
Creates a new directory named `devops_task`.

**Purpose:**  
Used to organize files related to the DevOps task inside a separate folder.

---

### 4. `cd devops_task`
**Description:**  
Changes the current directory to `devops_task`.

**Purpose:**  
Allows performing file operations inside the newly created directory.

---

### 5. `touch file1.txt`
**Description:**  
Creates an empty file named `file1.txt`.

**Purpose:**  
Demonstrates file creation using Linux commands.

---

### 6. `ls`
**Description:**  
Lists the contents of the `devops_task` directory.

**Purpose:**  
Confirms that `file1.txt` was successfully created.
